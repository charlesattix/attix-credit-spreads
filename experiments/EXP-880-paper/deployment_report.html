1. Strategy Summary

EXP-880 combines three validated innovations into a single deployable strategy:

ComponentSourceWhat It Does
ML Signal FilterEXP-8603-model ensemble (XGB + RF + ExtraTrees), P≥0.75 entry gate, quarterly retrained
Regime LeverageEXP-8402x base leverage, scaled by regime: bull 1.2x, sideways 0.8x, bear 0.3x
Crisis Hedge V2EXP-880Drawdown-controlled delevering (2%→7%), VIX-tiered reduction, put overlay at VIX>30

How It Works (Signal to Trade)

  1. Signal generation: 3-model ensemble scores each day's opportunity (0.0–1.0)
  2. Filter: Only signals with P ≥ 0.75 proceed (rejects ~40% of signals)
  3. Sizing: P≥0.90 → 100%, P≥0.80 → 75%, P≥0.75 → 50% of base size
  4. Leverage: Base 2x × regime multiplier × crisis hedge scale
  5. Execution: $5+ spread widths, limit orders, mid-day (10:30–14:00)
  6. Exits: 50% profit target, 14 DTE time exit, 1.5σ stop, VIX spike
