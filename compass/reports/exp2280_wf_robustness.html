Method

  • Source: cached sparse 7-stream frame from EXP-2200 build_streams (compass/cache/exp2280_v6_sparse.pkl).
  • Streams: exp1220, xlf_cs, xli_cs, gld_cal, slv_cal, vol_arb, v5_hedge (all REAL trade tapes / cached v3 pickle).
  • Walk-forward: 252 train / 63 test / step 63 → 20 folds. Causal — weights and vol scale fit ONLY on train.
  • Equal-risk weights = normalise(1 / σ_i) on train-window daily returns.
  • Vol target = 0.15 annualised; scale = 0.15 / (train_portfolio_σ × √252), fixed for the fold.
  • Per-fold Sharpe = mean(daily_ret) / std(daily_ret) × √252 on the 63-day test window.
  • Pooled OOS = concatenation of all 20 test windows, then same formula.
