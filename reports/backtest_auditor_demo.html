Backtest Auditor Demo — Catching Known Bugs

3 real-world audits demonstrating each bug class we've encountered
Why this matters: We spent weeks building strategies that looked great on synthetic data (EXP-880: Sharpe 4.97, CAGR 76.9%) only to discover they were bankrupt on real data (-104% return). The Sharpe formula bug inflated 3.76 → 9.09. This auditor catches these errors before they waste time.
